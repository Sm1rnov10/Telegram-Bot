# 🔐 Password Generator Bot

**🤖 Умный Telegram бот для генерации и проверки надёжности паролей**

[Попробовать бота](https://t.me/TgPythonPetProjectbot)

## 📋 О проекте

**Password Generator Bot** — это удобный Telegram бот для создания криптостойких паролей и проверки их надёжности. Проект с открытым исходным кодом, написанный на Python.

## ✨ Возможности

- **🔐 Генерация паролей** с настраиваемой длиной (4-64 символа)
- **⚙️ Гибкие настройки** — выбор типов символов (буквы, цифры, спецсимволы)
- **🔍 Проверка паролей** — оценка стойкости и времени взлома
- **🎯 Удобный интерфейс** — меню с кнопками, интерактивные настройки
- **💾 Сохранение настроек** — для каждого пользователя индивидуально

## 🛠 Технологии

- **Python 3.13** с библиотекой `pyTelegramBotAPI`
- **Криптостойкая генерация** — модуль `secrets`
- **Безопасность** — экранирование HTML, защита от XSS

## 🚀 Быстрый старт

```bash
# Клонировать репозиторий
git clone https://github.com/Sm1rnov10/Telegram-Bot.git
cd password-bot

# Создать виртуальное окружение (рекомендуется)
python -m venv venv
# Активировать:
# Windows: venv\Scripts\activate
# Linux/Mac: source venv/bin/activate

# Установить зависимости
pip install -r requirements.txt

# Создать файл .env с токеном
echo "BOT_TOKEN=ваш_токен_здесь" > .env

# Запустить бота
python main.py
```
